# Belajar Git #

1. Membuat Repository Local

    ```
    git init
    ```

2. Memindahkan perubahan file dari working area ke staging area

    ```
    git add namafile
    ```

3. Menyimpan perubahan yang ada di staging ke repository local

    ```
    git commit
    ```

    Atau

    ```
    git commit -m "Commit pertama"
    ```
